Projet python

Lors de ma première année de BTS SIO, j'ai travaillé en binôme sur le développement d'un outil en Python destiné à automatiser la création de bases de données personnalisées pour les utilisateurs d'une entreprise. Ce projet, réalisé sur une semaine, consistait à extraire les utilisateurs depuis Active Directory, à générer automatiquement les scripts SQL nécessaires (CREATE et DROP) et à les exécuter sur un serveur MariaDB.
