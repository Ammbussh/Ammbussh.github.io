Projet Android

Dans le cadre de ma deuxième année de BTS SIO, j'ai contribué, au sein d'une équipe de trois personnes, à la conception d'une application Android de réservation de restaurants. Complémentaire au site web R3st0.fr (réalisé lors d'un projet précédent), cette application permet aux utilisateurs de consulter la liste des restaurants, d'accéder aux informations détaillées et de réserver une table directement depuis leur smartphone.

Technologies

  • PHP / SQL / Java

Compétences

ressources numériques ▸Gérer des sauvegardes ▸Traiter des demandes concernant les applications Analyser les objectifs et les modalités d'organisation

Grâce a android studio j'ai pu concevoir cette application ce qui m'a permit de dévellopper mes compétences. Nous avons recensé et utilisé différentes ressources nécessaires au projet, comme Android Studio pour le développement, et GitLab pour la gestion de version puis de sauvegardes avec gitlab, Analyser les objectifs et les modalités d'organisation : Au début du projet, nous avons défini les objectifs fonctionnels de l'application (consultation de restaurants, réservation) ainsi que notre méthode de travail en équipe, en planifiant les tâches sur un tableau de bord partagé.
