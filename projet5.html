Projet Site Resto

Description du projet

Lors de ma deuxième année de BTS SIO, j’ai contribué à l’amélioration d’une première version du site R3st0.fr, au sein d’une équipe de trois personnes. S’inspirant de plateformes telles que TripAdvisor, ce site propose une base de données complète de restaurants ainsi que les avis des utilisateurs, afin de faciliter le choix des consommateurs.

Technologies

  • HTML / CSS / PHP / SQL

Compétences

▸Traiter des demandes Participer à l’ évolution d’ un site Web exploitant les données de l’ organisation.

J’ai utilisé plusieurs outils et technologies : NetBeans, GitLab, HTML, CSS, PHP et SQL. Ce projet m’a permis de renforcer mes compétences en développement web, gestion de projet collaboratif et déploiement. J’ai également appris à travailler en équipe en tant que Scrum Master et à gérer les tâches et l’équipe du mieux possible.
